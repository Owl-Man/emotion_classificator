# 🎭 Классификатор эмоций лица

Проект по классификации эмоций человека по фотографии лица с использованием моделей глубокого обучения, FastAPI и Streamlit.

## 📋 Описание проекта

Данный проект реализует полный цикл разработки ML-приложения:
- Сравнение нескольких моделей классификации эмоций
- Выбор и сохранение лучшей модели
- Развёртывание модели в виде REST API с помощью **FastAPI**
- Создание удобного пользовательского интерфейса с помощью **Streamlit**
- Деплой приложения на облачные платформы

### Используемый датасет
**Human Face Emotions** (Kaggle) — датасет с изображениями лиц людей, размеченных по эмоциональным состояниям.

**Классы эмоций (5 классов):**
- Angry (Злость)
- Fear (Страх)
- Happy (Радость)
- Sad (Грусть)
- Surprise (Удивление)

## 🚀 Раздел 2. Развёртывание API (Backend)

### Технологии:
- **FastAPI** — высокопроизводительный веб-фреймворк для создания API
- **TensorFlow 2.17** — загрузка и инференс модели
- **Uvicorn** — ASGI-сервер
- **CORS** — для корректной работы со Streamlit

### Основные эндпоинты:
- `GET /` — проверка работоспособности API
- `POST /predict` — классификация эмоции по загруженному изображению
